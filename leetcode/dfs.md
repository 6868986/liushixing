- #### [17. 电话号码的字母组合](https://leetcode.cn/problems/letter-combinations-of-a-phone-number/)

  ```java
  List<元素> res;
  void dfs(int i,int n,元素，依赖变量){
    if(i == n){
      res.add(元素);
      return;
    }
    for(){
      元素.add()
      dfs(i + 1,n,元素,依赖变量)；
      元素.remove(元素.length - 1);  
    }
  }
  ```
